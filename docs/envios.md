# Zonas y Políticas de Envío

Tenemos cobertura de envío a domicilio en las siguientes colonias:
- **Zona Centro**: Costo de envío $20 MXN. Tiempo estimado: 25-35 minutos.
- **Zona Sur (Colonia Roma, Condesa, Del Valle)**: Costo de envío $35 MXN. Tiempo estimado: 40-50 minutos.
- **Zona Norte (Lindavista, Vallejo)**: Costo de envío $45 MXN. Tiempo estimado: 45-60 minutos.

*El pedido mínimo para envíos a domicilio es de $150 MXN.*
