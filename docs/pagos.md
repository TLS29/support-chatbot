# Métodos de Pago Aceptados

Aceptamos las siguientes formas de pago:
1. **Efectivo**: Al recibir tu pedido en la sucursal o con el repartidor. (Por favor, indica si necesitas cambio al hacer el pedido).
2. **Tarjetas de Crédito y Débito**: Visa, MasterCard y American Express. Llevamos terminal inalámbrica a tu domicilio.
3. **Transferencia SPEI**: Solo disponible para pedidos mayores a $500 MXN.
